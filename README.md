# roddy-flatpaks
Personal Flatpak Repo

## Add repo
``` bash
flatpak remote-add roddy-flatpak https://kichirouhoshino.github.io/roddy-flatpaks/index.flatpakrepo
```
This repo relies on Flathub for its runtimes.
